Учебный backend-проект - REST API для управления спортивными тренировочными программами, реализованный на C# с использованием MongoDB.

## Предметная область
Система предназначена для хранения и управления тренировочными программами. Каждая программа описывает тренировочный план: название, описание, длительность в неделях и количество тренировок в неделю.

## Модель данных
- TrainingProgram  
- Id  
- Title  
- Description  
- DurationInWeek  
- CntInWeek

## Функциональность API
- создание тренировочной программы
- получение списка программ
- получение программы по ID
- обновление программы
- удаление программы
- хранение данных в MongoDB

## Технологии
- C# (.NET Web API)
- MongoDB
- REST API
- BSON-модели (MongoDB Driver / MongoDB .NET Driver)

## Ограничения
- отсутствует авторизация и пользователи
- упрощённая предметная модель (одна сущность)
- нет бизнес-логики уровня расписаний или связей между сущностями
- минимальная валидация данных
- проект учебный

## Замечания по требованиям
Предметная формировались в процессе реализации, поэтому итоговая модель предметной области представляет упрощённую версию системы.
